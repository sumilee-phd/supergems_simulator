실제 Super Gems 적용 방법
  1. 아래 "실제 슈퍼젬스 만들기" 클릭 → Gemini 페이지 이동
  2. 입력창에 앱 설명 붙여넣기 → 전송
  3. 생성된 단계에서 "Open Advanced Editor" → Opal 진입
  4. 각 노드 클릭 → 위 프롬프트를 해당 칸에 붙여넣기
  5. 우측 상단 Publish → 공유 링크 완성!
